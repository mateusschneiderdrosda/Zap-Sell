FAQ
Dúvidas frequentes
Preciso de conhecimento técnico?▼
Não. Tudo é configurado por painéis visuais, sem código.
O WhatsApp pode banir meu número?▼
O ZapSell usa tecnologia similar ao WhatsApp Web com limites seguros. Recomendamos número exclusivo para o bot.
Como funciona a integração com a Cakto?▼
Cada venda na Cakto dispara um webhook para o ZapSell, que salva o comprador e envia boas-vindas no WhatsApp em segundos.
Posso usar o plano gratuito para sempre?▼
Sim! O plano gratuito não expira. Inclui 50 mensagens/mês para testar. Para escalar, planos pagos a partir de R$97/mês.
Posso cancelar quando quiser?▼
Sim, sem multa. Cancele pelo painel ou pelo suporte. Você mantém o acesso até o fim do período pago.
